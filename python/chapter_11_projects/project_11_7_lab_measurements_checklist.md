# 11.7 HF-Messungen Schritt für Schritt (Checkliste)

## 1) LNA-Rauschmessung
- Gain grafisch darstellen
- Rauschboden messen
- Pluto-Eigenrauschen berücksichtigen

## 2) Filterkurven (Bandpass/Bandstop)
- Satsagen + Tracking-Generator
- Pluto TX → DUT → Pluto RX

## 3) Spektralanalyse von Modulationen
- AM/SSB/FM live untersuchen
- IQ-Diagramme erzeugen
- Echtzeit-Wasserfall

## 4) Intermodulationstest
- Zwei-Ton-Signal erzeugen
- IMD3 messen
- FFT-Analyse

## 5) TX-Sauberkeit / Nachbarkanäle
- DAC-Rauschen
- Spurious
- LO-Leakage
